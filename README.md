## About

Sailo is a modern messaging app designed for simple and seamless communication.

Built with Kotlin and Jetpack Compose, Sailo provides a clean and modern messaging experience with real-time conversations, user profiles, and a smooth Android-first interface.

## Features

* Real-time messaging
* User authentication
* User profiles
* Profile pictures
* Persistent login sessions
* Modern chat interface
* Smooth screen transitions
* Bottom navigation
* Clean and minimal design
* Cloud-powered user and messaging data

## Upcoming Updates

* Image sharing
* Voice messages
* Message replies
* Message reactions
* Read receipts
* Online status
* Group chats
* Push notifications
* Improved profile customization
* Performance improvements

## Built With

* Kotlin
* Jetpack Compose
* Material 3
* Firebase Authentication
* Cloud Firestore
* Cloudflare R2
* Coil

## Platform

Sailo is currently designed for **Android Phones**.

Support for more platforms may come in the future.
